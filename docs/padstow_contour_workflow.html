Contour Workflow

A simple QGIS guide for generating contour outputs from survey surfaces and preparing clean contour images for the monitoring app.

What The Contour Layer Is For

Interpretation

Contours help show surface shape, ridges, troughs, and the way sandforms are shifting through time.

App support

The app can use contour images as an optional layer in Compare and as supporting context for Highlight Change.

Step-by-Step QGIS Workflow

  1. Load the survey surface.

    Use the survey surface you want to contour.

  2. Open the contour tool.

    Search for Contour in the Processing Toolbox.

  3. Choose the contour interval.

    Use an interval that is detailed enough to show the surface shape without becoming cluttered.

    Example starting point: 0.5 m
  4. Set the elevation field name.

    Use a clear name such as ELEV.

  5. Run the contour output.

    This creates the contour line layer.

  6. Style the contour layer clearly.

    For app exports, keep the line styling simple and readable.

  7. Place the contour layer over the ortho or colour elevation view.

    Use the same rotated frame and export extent as the other app images if you want the layers to line up neatly.

  8. Export the contour image.

    Keep it as a clean overlay-style image if possible.

Styling Recommendations

Line colour

Use a consistent colour that stands out from the ortho or colour elevation background.

Line weight

Keep lines fine enough to avoid clutter but thick enough to survive export.

Labels

Use labels only if they genuinely help the interpretation. For the app, clean unlabeled lines are often easier.

Export frame

Use the same extent, rotation, and image size as the matching ortho or elevation view if the app needs them to sit together.

Good Habits

  • Use the same contour interval for comparable survey rounds unless there is a strong reason not to.
  • Keep export naming clear and consistent.
  • If you rotate the map for a landscape page or app view, export all related layers from that same rotated frame.
  • Use contours to support interpretation, not to replace the measured difference workflow for volume change.
